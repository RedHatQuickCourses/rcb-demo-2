#   Introducing RHEL Lightspeed for RHEL Systems

=== Introducing RHEL Lightspeed for RHEL Systems

RHEL Lightspeed brings generative AI capabilities to RHEL systems to assist with configuration, management, and troubleshooting.

*   It leverages link:https://www.ibm.com/watsonx[WatsonX AI API LLM^] (Large Language Model) as its underlying generative AI functionality.
*   The WatsonX model is deployed as a SaaS external infrastructure.
